# 🌊 Dive into the Ocean Depths

An immersive, multi-page storytelling web experience that takes users on a cinematic journey from the ocean surface to the mysterious abyss.

---

## 🧠 Inspiration

We often explore space more than our own oceans. This project is inspired by the mystery, beauty, and depth of the ocean — designed to create an emotional and interactive storytelling experience that reveals different underwater zones as the user dives deeper.

---

## ✨ Features

* 🌐 **Multi-Page Experience** – Each page represents a different ocean depth zone
* 🎬 **Cinematic Storytelling** – Smooth transitions and immersive UI
* 🌊 **Scroll-Based Interactions** – Parallax effects and animated reveals
* 💡 **Interactive Elements** – Hover effects, glow animations, and dynamic UI
* 🎨 **Modern UI/UX** – Glassmorphism, gradients, and dark immersive theme
* 📱 **Fully Responsive** – Optimized for desktop, tablet, and mobile devices
* 🔊 **Ambient Experience (Optional)** – Sound toggle for underwater feel

---

## 🧭 Pages Overview

### 1. Home (Surface Level)

* Entry point with ocean surface visuals
* CTA: “Start Diving”

### 2. Sunlight Zone (0m–200m)

* Bright underwater life
* Coral reefs and fish

### 3. Twilight Zone (200m–1000m)

* Dim lighting with glowing creatures
* Bioluminescence effects

### 4. Midnight Zone (1000m–4000m)

* Dark, mysterious environment
* Deep-sea creatures

### 5. Abyss Zone (4000m+)

* Minimal light, maximum mystery
* Final emotional message

### 6. About Page

* Information about ocean layers
* Educational insights

## ⚙️ Installation & Setup

```bash
# Clone the repository
git clone https://github.com/your-username/ocean-depths.git

# Navigate into the project
cd Ocean-Depths
cd Frontend

# Install dependencies
npm install

# Start the development server
npm run dev
```
